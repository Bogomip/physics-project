How long (time) does it take for the meteor to come to a complete stop?

Now we start to get into some basic motion content. So we have to satisfy a basic equation here. We have:

  • Acceleration
  • Net Force
  • Mass
  • Normal and Weight Forces
  • Velocity

Given we know both acceleration and the initial velocity, as well as the final velocity (0 m/s) we can use:

$$ acceleration = \frac{{'{'}} change in velocity {{'}'}}{{'{'}} time {{'}'}}$$ s

Rearranging gives:

$$ time = \frac{{'{'}} change in velocity {{'}'}}{{'{'}} acceleration {{'}'}}$$ m/s2
$$ time = \frac{{'{'}} {{ calculateChangeInVelocity() }} {{'}'}}{{'{'}} {{ calculateHorizontalAcceleration() }} {{'}'}} = {{ calculateTimeToStop() }}$$ s
How much distance did the meteor cover while sliding to a stop?

Finally for this question we have to calculate the distance travelled, but we have everything we need!

  • Time the object travelled along the ground
  • The change in velocity
  • The fact that the acceleration on the object was constant

This last point is important because it means we can say that:

$$ V_{{'{'}}final{{'}'}} = V_{{'{'}}average{{'}'}} * 2$$ m/s

Thus if our velocity starts at {{ initialVelocity }} m/s and ends at {{ finalVelocity }} m/s we can see that:

$$ V_{{'{'}}average{{'}'}} = \frac{{'{'}} V_{{'{'}}final{{'}'}} {{'}'}}{{'{'}} 2 {{'}'}} = {{ calculateAverageVelocity() }}$$ m/s

And as we know the disdtance is just the product of time and average velocity:

$$ distance = average speed * time = {{ calculateAverageVelocity() }} * {{ calculateTimeToStop() }} = {{ calculateDistanceToStop() }}$$ meters
